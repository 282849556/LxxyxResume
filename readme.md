# Introduction

### 简历为仿写，原简历页面地址: [张秋怡 - 个人简历](https://joyeecheung.github.io/resume/)

简历基于Gulp+Scss+Jade。  
访问地址：[Lxxyx的简历](http://115.159.148.159/)

### 运行方式：
替换info.json，运行gulp即可。
需要自动部署的，运行`gulp deploy`即可，需要在gulpfile中更改为你的地址（！会抹去你github pages内容，慎用！）
